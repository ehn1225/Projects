# 자본 관리 프로그램 Ver2.0

### 이 프로그램은 수입 및 지출을 관리하는 가계부 프로그램입니다.
- 월 소득을 바탕으로 수입/지출을 관리합니다.
- 잔여 자본이 얼마나 남았는지 비율로 보여줍니다.
- 개발 일자 : 2015.12

## 기능
- 지출 기록 기능(현금/카드 설정)
- 통장 입금/출금 기록 기능
- 수입이 발생한 경우 수입 기록 가능
- 가계부 리셋 기능

## Ver 1.0과 차이점
- 통장 입금/출금 기능 추가
- 기존에는 추가 수입의 경우 기록이 남지 않았던 것을, 수입도 기록에 남기도록 함
